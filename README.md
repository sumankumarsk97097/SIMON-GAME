# 🎮 Simon Game

A simple and interactive Simon Game built using HTML, CSS, and JavaScript.

## 🚀 Features
- Interactive UI
- Random color sequence generation
- Increasing difficulty levels
- Level tracking
- Score tracking
- Keyboard controls

## 🛠️ Technologies Used
- HTML
- CSS
- JavaScript

## ▶️ How to Play
1. Press any key to start the game
2. Watch the color pattern carefully
3. Repeat the same sequence by clicking the buttons
4. The sequence gets harder after every level
5. Wrong input will end the game
